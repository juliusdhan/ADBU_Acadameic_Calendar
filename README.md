# 📅 ADBU Academic Calendar Portal

## 📖 Project Description

The **ADBU Academic Calendar Portal** is a web-based system designed to manage and display academic events for **Assam Don Bosco University**.  

It allows students and faculty to easily view important dates such as semesters, exams, holidays, and deadlines, while administrators can manage events efficiently through a dedicated admin panel.

---

## ✨ Features

- 🔐 User & Admin Authentication (Login / Register)
- 📊 Dashboard with event statistics
- 📅 Interactive Calendar View
- 📋 All Events Listing (Responsive table & cards)
- ➕ Add Event (Admin only)
- 🗑 Delete Event (Admin only)
- 🌙 Dark Mode & ☀ Light Mode Toggle
- 📱 Fully Responsive Design

---

## ⚙️ Prerequisites

Make sure you have the following installed:

- PHP (>= 8.0)
- MySQL
- XAMPP / WAMP / LAMP (Local Server)
- Web Browser (Chrome, Edge, etc.)
- Git (optional but recommended)

---

## 🚀 Installation Steps

Follow these steps to run the project locally:

### 1. Clone the Repository

git clone https://github.com/juliusdhan/ADBU_Acadameic_Calendar.git


### 2. Move to Project Folder

cd adbu-academic-calendar

### 3. Move Project to Server Directory
If using XAMPP:
Move folder to: C:\xampp\htdocs\

### 4. Start Server
Open XAMPP
Start Apache and MySQL

### 5. Create Database
Open phpMyAdmin
Create a database (e.g., academic_calendar)
Import the provided SQL file

### 6. Configure Database Connection

Edit:
config/db.php  OR  includes/db.php

### 7. Run the Project
Open browser:

http://localhost/Adbu_Acadamic_calendar
